# Sirano

Network trace and log file anonymizer.

This version is for testing purposes.

## Features

### Supported file types

* Pcap packet-capture
* Text log file

### Supported protocols

* ARP
* DNS <sup>[new]</sup>
* Ethernet
* ICMP <sup>[new]</sup>
* IP
* RTP <sup>[new]</sup>
* SDP
* SIP
* TCP
* UDP

### Supported data types

* MAC address
* IPv4 address
* Domain name
* Name
