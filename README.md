# Sirano

Network trace and log file anonymizer.

## Features

### Supported file types

* Pcap packet-capture
* Text log file

### Supported protocols

* ARP
* Cooked Linux
* DNS
* Ethernet (802.3)
* Ethernet II
* ICMP
* IP
* LLC
* RTP
* SDP
* SIP
* TCP
* UDP

### Supported data types

* MAC address
* IPv4 address
* Domain name
* Name
